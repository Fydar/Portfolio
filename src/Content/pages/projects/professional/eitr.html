An indie-hit 2d pixel-art dungeon crawler.

I joined the indie title ‘Eitr’ team as a remote programmer, working on many of the titles core systems.

These systems often feature data-driven design utilising Unity's ScriptableObjects. I worked on many of the core systems including:

  • Items and inventory system
  • Item and generic tooltips
  • Character stats
  • Skill trees and skill tree editors
  • Input mapping and control schemes
  • Achievements (to support PS4 achievements)

Programming on Eitr inspired my ‘RPGCore’ project, which is now an open-source project.

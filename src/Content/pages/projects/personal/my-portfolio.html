This portfolio website has been created using ASP.NET Core.

Patio

I created a CSS and JavaScript package called 'Patio' to power my portfolio website.

'Patio' initially drew from a lot of the concepts implemented by Bootstrap, as I have became familar with Bootstrap during my time working at TT Games.

Progressive Loading

My portfolio implements pre-compressed and minified images for the purposes of showing low-quality previews whilst the page is loaded.
